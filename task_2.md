## Check-list for [School21 study methodology](https://21-school.ru/methodology)

| № | Проверка | Описание проверки | Статус |
| --- | --- | --- | --- |
| 1 | Отображение текста | Проверка корректности текста, отсутствие ошибок, обрезания и клиппинга | |
| 2 | Структура раздела | Проверка логичности и удобства расположения блоков и их последовательности | |
| 3 | Отображение изображений | Проверка корректной загрузки изображений и их отображение | |
| 4 | Кликабельность элементов | Проверка работы кнопок и ссылок внутри раздела | | 
| 5 | Адаптивность страницы к разрешению и масштабу | Проверка корректного отображения на различных разрешениях экрана, включая мобильные устройства | | 
| 6 | Читаемость текста | Проверка контрастности и удобства восприятия текста | |
| 7 | Загрузка страницы | Проверка скорости и корректность загрузки раздела | |
